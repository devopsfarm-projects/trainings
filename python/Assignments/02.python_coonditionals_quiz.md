
### Multiple Choice Questions (5)

1. What will this code output?

   ```python
   x = 7
   if x % 2 == 0:
       print("Even")
   else:
       print("Odd")
   ```

   A) Even
   B) Odd
   C) 7
   D) Error

2. Which of these is a valid if-elif-else structure in Python?
   A) `if x > 10 then:`
   B) `if (x > 10):`
   C) `if x > 10:`
   D) `if x > 10 {}`

3. What does `elif` stand for?
   A) Else if
   B) Else loop
   C) Error list
   D) End line

4. What happens if there’s no `else` after an `if` condition and the condition is False?
   A) Error
   B) Skips the if block
   C) Executes default block
   D) None of the above

5. What is the output?

   ```python
   x = 5
   if x:
       print("True")
   else:
       print("False")
   ```

   A) True
   B) False
   C) Error
   D) None
